# miniC compiler

用Rust实现的SysY to RISC-V编译器。

一边学习Rust一边完成这个项目,可能有很多不地道的写法,我还在修正中。

前端从SysY生成Koopa IR,后端从Koopa IR生成RISC-V。

前端部分暂时使用文本形式IR,或许暑假会改成内存形式。
